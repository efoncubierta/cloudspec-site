## Members
* **local_gateway_id**
(`string`):
The ID of the local gateway.
* **outpost_arn**
(`string`):
The Amazon Resource Name (ARN) of the Outpost.
* **owner_id**
(`string`):
The ID of the AWS account ID that owns the local gateway.
* **region**
(`string`):
The AWS region.
* **state**
(`string`):
The state of the local gateway.
* **tags**
(`key_value[]`):
The tags assigned to the local gateway.
